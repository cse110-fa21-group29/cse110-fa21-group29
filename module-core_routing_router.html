    (inner) navigateFromUrl(url)

    Navigate to a page based on only the URL. Used when we are navigating without a state. If we can match the url to a route pattern, we load the route onto the page and store its state in the current history entry. If we cannot match the url to a route pattern, we redirect to the home page.
